# React Floating Label

MDC React Floating Label acts as a React container around MDC Floating Label. Please see [MDC Fab](https://github.com/material-components/material-components-web/tree/master/packages/mdc-floating-label).

## Installation

```
npm install @material/react-floating-label
```


## Props

Prop Name | Type | Description
--- | --- | ---
className | String | Classes to be applied to the root element.
